# Productos Service

Microservicio desarrollado con Spring Boot para la gestión de productos.

## Tecnologías utilizadas

- Java 21
- Spring Boot 3
- Spring Data JPA
- H2 Database
- JUnit 5
- Mockito
- Maven

## Funcionalidades

- Crear productos
- Buscar productos por ID
- Actualizar stock
- Eliminar productos
- Validaciones de negocio

## Suite de pruebas implementada

Se implementaron pruebas unitarias para:

- Casos exitosos
- Casos de error
- Validaciones de negocio
- Pruebas parametrizadas
- Uso de mocks con Mockito
- Verificación de interacciones
- Captura de argumentos con ArgumentCaptor

## Resultado de pruebas

Ejecutar:

```bash
mvn test
```
